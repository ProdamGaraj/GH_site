## Чеклист материалов от заказчика

### Обязательные (блокируют старт разработки)

- [ ] **Брендбук** или гайдлайн (цвета, шрифты, логотип)
- [ ] **Логотип** в форматах: SVG (вектор), PNG (прозрачный фон)
- [ ] **Доступ к CRM** (тестовый аккаунт или sandbox)
- [ ] **Документация API CRM** (или контакт ответственного)
- [ ] **Список проектов** (ЖК и коммерция) с базовой информацией
- [ ] **Контактные данные** для сайта (телефон, email, адрес)
- [ ] **Домен** — подтверждение владения и доступ к DNS

### Важные (нужны к концу Sprint 1)

- [ ] **Фотографии объектов** — фасады, интерьеры, МОПы, благоустройство
- [ ] **Рендеры** — если объекты ещё строятся
- [ ] **Планировки квартир** — изображения в хорошем качестве
- [ ] **Тексты** — О компании, описания проектов
- [ ] **Доступы к аналитике** — Яндекс.Метрика, Google Analytics
- [ ] **Доступ к чат-сервису** — если используется

### Желательные (можно позже)

- [ ] **Фото команды** — для раздела "О компании"
- [ ] **Отзывы клиентов** — тексты, фото, видео
- [ ] **Сертификаты/награды** — для раздела "О компании"
- [ ] **Логотипы партнёров/банков**

## Риски Sprint 0

**Риск 1: Нет доступа к CRM API**
- Вероятность: Высокая
- Влияние: Критичное
- Митигация: Запросить в первый день, эскалировать

**Риск 2: Нет брендбука**
- Вероятность: Средняя
- Влияние: Высокое
- Митигация: Согласовать минимум: цвета + шрифт + лого

**Риск 3: Заказчик долго отвечает**
- Вероятность: Средняя
- Влияние: Среднее
- Митигация: Установить SLA на ответы (24-48ч)

**Риск 4: Неясные требования к контенту**
- Вероятность: Средняя
- Влияние: Среднее
- Митигация: Подготовить примеры и шаблоны
